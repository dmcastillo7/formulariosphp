## Descripción del proyecto

Este proyecto consiste en el desarrollo de una interfaz web básica que incluye un formulario de inicio de sesión (Login), un formulario de registro (Register) y un formulario de ventas, utilizando únicamente HTML, PHP y CSS.
El objetivo es aplicar conceptos fundamentales de diseño web, estructura semántica y estilos modernos para mejorar la experiencia del usuario.

## El sistema permite:
* Iniciar sesión desde una pantalla de login.

* Redirigir al formulario de ventas mediante un botón de registro.

* Visualizar un formulario de ventas organizado por paquetes, pedidos y sección de pago.
